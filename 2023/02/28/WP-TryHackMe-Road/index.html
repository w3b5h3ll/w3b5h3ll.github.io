当你在一个使用 sudo 的系统上登录时，你可能会被授予一些权限，这些权限可能允许你执行特定的操作，例如安装软件包、编辑系统文件等。使用 sudo -l 命令，你可以查看自己当前的权限，以及哪些命令可以使用 sudo 执行。

LD_Preload提权

env_keep 是一个在 Linux 系统中用于控制哪些环境变量可以在 sudo 命令执行期间保持不变的设置。

当用户使用 sudo 命令来执行需要特权的任务时，sudo 会创建一个新的 shell 环境来执行该命令。在这个新的 shell 环境中，sudo 可以使用 env_keep 设置来保持一些环境变量的值不变，而不是使用默认的环境变量值。

在 C 语言中，_init 函数是一个特殊的函数，用于在共享库加载时执行一些初始化操作。当使用共享库时，操作系统会自动加载共享库并在加载时调用 _init 函数。
